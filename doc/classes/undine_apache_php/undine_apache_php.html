Class: undine_apache_php

The undine_apache_php class is responsible for the package management, installation, and configuration of the php53 module for httpd in Undine.

It should not be necessary to declare this class directly, as it will be declared automatically by the undine class, which all Undine sites should use.

Provisioning for Apache and PHP are managed by undine_apache and undine_php, respectively.
